# project_react_native
In the world of mobile applications, the use of cross-platform frameworks is becoming increasingly popular due to the ability to enable the development of apps for different platforms using a single code base. In this study, a comparative performance benchmark of the main cross-platform solutions available, such as Flutter and React Native, will be realized. Execution speed, ease of development, and level of access to native device functionality will be analyzed. In addition, the stability of the built applications based on different parameters will be analyzed. An implementation of an application using both technologies will also be presented, which will be used for testing. The result of this study will allow us to provide an overview of the performance of the main cross-platform frameworks for mobile app development.
