# Abstract
In the web application landscape, cross-platform frameworks are becoming increasingly popular due to the ability to develop apps for different platforms using a single code base. This study focuses on the performance comparison between Flutter and React Native in cross-platform web application development. Results of code execution in both client-side and server-side rendering will be analyzed, evaluating execution speed, ease of development, and access to device-native features. In addition, a practical demonstration will be presented with the implementation of an application using both technologies, which will be used to test performance through benchmarks. The results of this study will provide an overview of the two most popular cross-platform frameworks and their performance in web application development.
